# Trans-Time AI Prompts

Коллекция системных промтов для интеграции ИИ в логистическую платформу **Trans-Time.ru**.

**Модули:**
- `core.yml` — базовая идентичность и стиль ИИ.
- `map.yml` — интеллект интерактивной карты (маршруты ГВТ/Н3, ограничения, объяснения).
- `chat.yml` — общение и диалоговые паттерны.
- `ui.yml` — короткие UX-подсказки/аннотации.
- `dev.yml` — отладка, диагностика и безопасные дефолты.
- `test-suite/` — эталонные запросы/ответы для регрессионных тестов промтов.

## Использование
1. Выбери модель (например, `gpt-5`).
2. Передай `core.yml` как System Prompt, далее — профиль конкретного модуля (`map.yml`) как High-priority Instruction.
3. Проверяй корректность поведений через `prompts/test-suite/prompt-tests.json`.
